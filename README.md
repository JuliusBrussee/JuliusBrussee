### [Blueprint](https://github.com/JuliusBrussee/blueprint) — Specification-driven development for AI agents

A Claude Code plugin that turns natural language into blueprints, blueprints into parallel build plans, and build plans into working software — with automated iteration, validation gates, and dual-model adversarial review via Codex. The premise: AI agents without a specification have nothing to validate against. Blueprint gives them a contract.

```
Your idea → /bp:draft → Blueprints → /bp:architect → Build site → /bp:build → Working software
```

9 specialized agents. 15 deep-dive skills. Parallel execution across git worktrees. Cross-model peer review.

### [Caveman](https://github.com/JuliusBrussee/caveman) — Cut 75% of tokens by talking like a caveman

A Claude Code skill born from the viral observation that caveman-speak dramatically reduces LLM output tokens without losing any technical substance. One-line install. Same fix, 75% fewer words.

> **Normal Claude (69 tokens):** "The reason your React component is re-rendering is likely because you're creating a new object reference on each render cycle..."
>
> **Caveman Claude (19 tokens):** "New object ref each render. Inline object prop = new ref = re-render. Wrap in `useMemo`."

```bash
npx skills add JuliusBrussee/caveman
```

`Claude Code` `Skill` `MIT`

## Private & In-Progress Work

### [Stacklink](https://github.com/JuliusBrussee/stacklink) — AI memory layer for your documents

A RAG-powered knowledge system that hooks into Google Drive, builds a vector index with pgvector, and lets you query your own documents through a web UI or a Slack slash command. Ask a question, get an answer with citations pulled from your actual files. Supabase Auth + Google OAuth for a seamless connect-and-go experience.

The interesting part: real-time document sync, chunked embedding pipeline, and a Gemini-powered answer engine that actually tells you *where* it found the answer.

`TypeScript` `Next.js` `Supabase` `pgvector` `Gemini`
