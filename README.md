## What I'm Building

### [Blueprint](https://github.com/JuliusBrussee/blueprint) — specification-driven development for AI agents

A Claude Code plugin that turns natural language into blueprints, blueprints into parallel build plans, and build plans into working software. Automated iteration loops, validation gates, cross-model peer review, and a tmux-based launcher that runs multiple agents simultaneously on separate git worktrees.

```
Your idea → /bp:draft → Blueprints → /bp:architect → Build site → /bp:build → Working software
```

The premise: AI agents without a specification have nothing to validate against. Blueprint gives them a contract to build from and a standard to meet.

### [Revu](https://github.com/JuliusBrussee/revu-swift) — macOS-native study system

A local-first study app built in Swift/SwiftUI for turning course material into structured, durable learning. Spaced repetition, clean architecture, designed for serious academic use.

### [osxphotos-mcp](https://github.com/JuliusBrussee/osxphotos-mcp) — MCP server for macOS Photos

End-to-end Model Context Protocol integration for the macOS Photos library. Tools for finding, exporting, and explaining photo matches — privacy-by-default (no image bytes, GPS redacted).

### [The Prompt Library](https://github.com/JuliusBrussee/the-prompt-library) — production-ready prompts & schemas

Curated prompts, JSON schemas, an MCP server, and documentation for working with LLMs in production.

### [Christofides' Algorithm](https://github.com/JuliusBrussee/Christofides-Algorithm) — TSP approximation with visualization

Classic 1.5-approximation algorithm for the Traveling Salesman Problem, implemented with step-by-step visualization.
